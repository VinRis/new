# **App Name**: FarmSync

## Core Features:

- Livestock Selection: Welcoming screen to select livestock: Dairy, Poultry, Pigs, or Goats/Sheep.
- Dashboard KPIs: Display key performance indicators for the selected livestock.
- Record Management: Add/edit/delete production, financial, and health records for livestock.
- Finance Tracker: Manage all transactions. Each record should be editable and filterable by date/month.
- Upcoming Events Notification: Send notifications for upcoming health events.
- Farm Insights: Insights section offering a personalized and actionable suggestion for the farm based on data analysis. Uses AI to provide farmers with optimal strategies for improved results. For example, the AI tool would monitor records and determine the perfect time to buy/sell livestock, recommending actionable steps.
- Cloud Sync and Backup: Firestore integration for cloud data backup and sync.

## Style Guidelines:

- Primary color: Earthy green (#6B8E23), evoking a sense of nature and growth.
- Background color: Very light desaturated green (#F5F5DC) providing a neutral backdrop.
- Accent color: Muted gold (#B8860B) for highlights and actionable items, reminiscent of harvest.
- Body font: 'PT Sans' sans-serif for clear and modern body text.
- Headline font: 'Playfair' serif for elegant headings and titles; pairing with 'PT Sans' for body text.
- Use custom icons for each livestock category and record type, ensuring clarity and ease of use.
- Modern navigation bar at the bottom with a central FAB for adding records. Each navigation section (history,finance,health,report etc.) can easily be accessed.
